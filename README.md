# World Bank Indicator Dictionary

This project downloads the World Bank indicator catalog and builds:
- a searchable indicator catalog
- a dictionary keyed by indicator code
- helper functions for lookup and keyword search

## Features
- fetch full indicator metadata from the World Bank API
- save catalog as CSV
- save indicator dictionary as JSON
- search indicators by keyword
- retrieve code-to-name and code-to-metadata mappings
